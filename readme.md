## 一些值得考虑的问题

对于一个人有多种称呼，如何匹配?

人名的简称 例如 云天明，天明
附加职称等称谓 例如：汪淼，汪教授，汪淼教授,汪总
附加亲戚关系的 例如：杨母
附加年龄信息的 例如：苏小姐，赵先生，方老太太
